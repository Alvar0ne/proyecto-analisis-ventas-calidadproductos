# Proyecto de analisis de ventas y calidad de productos

Proyecto de análisis de ventas, calidad y devoluciones integrando 3 fuentes de datos distintas: ventas desde ERP en CSV, devoluciones desde Excel del área de postventa y catálogo maestro en JSON. Se realizó limpieza y estandarización de datos con Python, modelado relacional y consultas analíticas en SQL, y visualización ejecutiva en Power BI para identificar productos críticos, pérdidas por fallas, tasa de devolución y oportunidades de mejora operativa.
