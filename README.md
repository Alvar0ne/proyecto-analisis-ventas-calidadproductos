# proyecto-analisis-ventas-calidadproductos
Proyecto de análisis de ventas, calidad y devoluciones integrando 3 fuentes de datos distintas: ventas desde ERP en CSV, devoluciones desde Excel del área de postventa y catálogo maestro en JSON. Se realizó limpieza y estandarización de datos con Python, modelado relacional y consultas en SQL, y visualización ejecutiva en Power BI.
